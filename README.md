# CRM Data Warehouse for Business Insights

## 📌 Objective
To design and implement a CRM Data Warehouse that helps businesses analyze customer behavior, sales trends, and improve decision-making.

## 🧱 Architecture
- Source: CRM data (customers, orders, products)
- ETL: Python (data cleaning & transformation)
- Storage: SQL Data Warehouse
- Analysis: SQL + Python

## ⚙️ Tools Used
- Python (Pandas)
- MySQL
- Power BI / Tableau (optional)

## 📊 Features
- Customer segmentation
- Sales analysis
- Revenue trends
- Top customers & products

## 🚀 How to Run
1. Run SQL scripts in `/sql`
2. Run ETL script in `/etl`
3. Run analysis in `/analysis`
